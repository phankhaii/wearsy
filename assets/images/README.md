# Hình ảnh sản phẩm

Thư mục này chứa các hình ảnh sản phẩm và hình ảnh placeholder.

## Placeholder Image

Để sử dụng hình ảnh placeholder, bạn có thể:
1. Tạo một file `placeholder.jpg` với kích thước 400x400px
2. Hoặc sử dụng các dịch vụ placeholder online như:
   - https://via.placeholder.com/400x400
   - https://placehold.co/400x400

## Upload Images

Khi upload hình ảnh sản phẩm, hãy đảm bảo:
- Kích thước tối đa: 2MB
- Định dạng: JPG, PNG, GIF
- Tỷ lệ khuyến nghị: 1:1 (hình vuông)
- Độ phân giải tối thiểu: 800x800px

## Thư mục Upload

Hình ảnh được upload sẽ được lưu trong thư mục `uploads/` ở thư mục gốc.
